入门提示

  • 坐标系遵循 REP-103：+X 前，+Y 左，+Z 上。
  • 所有尺寸单位为 米 (m)，保留 3 位小数。
  • 每次保存前会生成 .bak，并保留最近 3 版。
  • 想看效果：在终端执行 roslaunch myrobot remote_rviz.launch。
  • 首次开机会请求连接串口，纯修改模型不需要串口。
  • 遇到“路径不允许”：说明尝试写到 myrobot 包之外，已被服务器拒绝。
