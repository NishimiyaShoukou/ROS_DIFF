# ROS_DIFF v2 Serial Protocol

协议版本：`2.1`

波特率：`115200 8N1`

字节序：所有 16/32 位整数均为 little-endian。

校验：CRC-8/MAXIM。

## CRC-8/MAXIM

| 参数 | 值 |
| --- | --- |
| Polynomial | `0x31`，反射实现常写作 `0x8C` |
| Init | `0x00` |
| RefIn / RefOut | `true / true` |
| XorOut | `0x00` |
| Check | `CRC("123456789") = 0xA1` |

帧头 `55 AA`、CRC 字节本身、上行帧尾 `0D 0A` 不参与 CRC。

## ROS To STM32

固定 11 字节。

| Byte | 内容 |
| --- | --- |
| 0..1 | 帧头 `55 AA` |
| 2 | 类型 `05` |
| 3..4 | 左轮目标输出轴转速，`rpm * 100`，`int16` |
| 5..6 | 右轮目标输出轴转速，`rpm * 100`，`int16` |
| 7..8 | 预留角度字段，当前为 0，`int16` |
| 9 | 控制标志 |
| 10 | CRC-8/MAXIM，覆盖 byte 2..9 |

STM32 只有在类型和 CRC 都正确时才刷新速度命令。命令超时后底盘会自动停车。

## STM32 To ROS

固定 25 字节。

| Byte | 内容 |
| --- | --- |
| 0..1 | 帧头 `55 AA` |
| 2 | 类型 `05` |
| 3..4 | 左轮实测输出轴转速，`rpm * 100`，`int16` |
| 5..6 | 右轮实测输出轴转速，`rpm * 100`，`int16` |
| 7..10 | 左轮累计编码器计数，`int32` |
| 11..14 | 右轮累计编码器计数，`int32` |
| 15..16 | IMU yaw，`degree * 100`，`int16` |
| 17 | 状态位 |
| 18..19 | 电池总电压，mV，`uint16` |
| 20 | 电量估算百分比，0..100 |
| 21 | 电池状态 |
| 22 | CRC-8/MAXIM，覆盖 byte 2..21 |
| 23..24 | 帧尾 `0D 0A` |

## Status Bits

| 位 | 含义 |
| --- | --- |
| bit0 | IMU 数据有效 |
| bit1 | STM32 正在上位机遥控模式 |
| bit2 | STM32 最近一次 IMU 读取存在错误 |
| bit3 | STM32 正在本地自动测试模式 |

ROS 仅在 bit0 有效时使用 IMU yaw。bit0 失效时自动退回纯双轮编码器里程计。

## Battery State

| 值 | 含义 |
| --- | --- |
| 0 | 正常 |
| 1 | 低电量 |
| 2 | 严重低电量 |
| 3 | 过压 |
| 4 | 不可用或 ADC 失败 |

ROS 发布 `/battery_state`，类型为 `sensor_msgs/BatteryState`。百分比来自电压线性估算，只适合告警和粗略观察，不适合作为精密容量计。

## Compatibility

v2.1 ROS 节点只接收 25 字节 STM32 上行反馈，新固件只接收 11 字节带 CRC 的下行命令。旧版无 CRC ROS 节点和旧版 STM32 固件不能混用。
